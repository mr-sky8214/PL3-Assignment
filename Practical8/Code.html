Find the sum of the series 1+11+111+1111+... upto n terms

C++ code for given problem statement

            // C++ program to find the sum of 
            // the series 1+11+111+1111+.... 
            #include  
            using namespace std; 
              
            // Function for finding summation 
            int summation(int n) 
            { 
                int sum = 0, j = 1, i; 
                for (i = 1; i <= n; i++) { 
                    sum = sum + j; 
              
                    // Appending a 1 at the end 
                    j = (j * 10) + 1; 
                } 
              
                return sum; 
            } 
              
            // Driver Code 
            int main() 
            { 
                int n = 5;
                cout << " " <<  summation(n); 
                return 0; 
            }

Sample Inputs and Outputs...

            Input : 3    
            Output : 1 + 11 + 111 +....
            Total sum is : 123

        
            Input : 4
            Output : 1 + 11 + 111 + 1111 +..... 
            Total sum is : 1234

        
            Input : 7
            Output : 1 + 11 + 111 + 1111 + 11111 + 111111 + 1111111 +..... 
            Total sum is : 1234567

Complexity

The above program iterates n2 n times

So time complexity is O(n)
